# 🧠 Image Classifier using PyTorch & Gradio

## 📌 Overview
This project is an AI-powered Image Classification web application developed using PyTorch and Gradio, and deployed on Hugging Face Spaces. The application uses a pretrained ResNet18 deep learning model to classify uploaded images and display top predictions with confidence scores.

---

## 🚀 Features
- Upload any image
- Predict image categories using AI
- Displays top predictions with confidence
- Interactive Gradio web interface
- Live deployment on Hugging Face Spaces

---

## 🛠️ Technologies Used
- Python
- PyTorch
- Torchvision
- Gradio
- Hugging Face Spaces

---

## 📷 Demo

Upload an image and the model predicts the object category.

Example Predictions:
- Egyptian cat
- Tiger cat
- Tabby

## 📖 Project Summary
This project implements an AI-powered Image Classification system using Deep Learning and Computer Vision techniques. The application is developed with PyTorch and deployed using Gradio on Hugging Face Spaces, enabling users to upload images through an interactive web interface and receive real-time predictions with confidence scores.

The system utilizes the pretrained ResNet18 convolutional neural network model capable of identifying a wide variety of real-world objects and image categories.
